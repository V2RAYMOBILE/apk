Инструкция по сборке APK для V2Ray VPN

Необходимые компоненты

  1. Flutter SDK (версия 3.x или новее)
  2. Android Studio с SDK
  3. Java JDK (версия 11 или новее)

Шаги по сборке APK

1. Клонирование проекта с Replit

git clone https://github.com/username/v2ray-vpn.git
cd v2ray-vpn

2. Обновление зависимостей

flutter pub get

3. Проверка настройки Flutter

flutter doctor

Убедитесь, что все зависимости установлены и Flutter правильно настроен для разработки Android-приложений.

4. Сборка APK

Для отладочной версии:

flutter build apk

Для релизной версии:

flutter build apk --release

APK-файл будет доступен по пути:

build/app/outputs/flutter-apk/app-release.apk

Часто задаваемые вопросы

Как установить приложение?

Скачайте APK-файл и откройте его на устройстве. При необходимости разрешите установку из неизвестных источников в настройках безопасности вашего устройства.

Какие протоколы поддерживаются?

Приложение поддерживает множество протоколов, включая TCP, WebSocket, gRPC и HTTP. Для каждого протокола реализована фрагментация для обхода блокировок.

Как работает автоматический выбор сервера?

Приложение автоматически проверяет пинг до всех доступных серверов и выбирает тот, который обеспечивает наиболее быстрое соединение.

Необходимо ли рутирование устройства?

Нет, приложение не требует рут-прав для работы и использует стандартный Android VPN API.

Как добавить свой собственный сервер?

В настройках приложения перейдите в раздел "Серверы" и нажмите кнопку "Добавить". Затем введите параметры подключения или импортируйте их через VLESS-ссылку.
